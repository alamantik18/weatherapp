# weatherprogram
Программа создана для быстрого просмотра минимальной и максимальной температуры любом городе мира.

Для того чтобы упаковать программу, установим библиотеку pyinstaller
pip install pyinstaller 

переходим в директорию проекта и прописываем команду:
pyinstaller -i"абсолютный путь к иконке" main.py
